# Project Setup

This guide explains how to install dependencies and run the application locally.

---

## Instructions

### 1. Install dependencies

Install the required project dependencies:

```bash
npm install
```

### 2. Run the app and payment provider servers concurrently

Start the development server:

```bash
npm run dev
```

### 3. Open the application

Open the application in your browser to view and test:
NextJs App: http://localhost:3000
Payment Provider: http://localhost:3001

---

## Running Tests

To run the test suite:

```bash
npm test
```
